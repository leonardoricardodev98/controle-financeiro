💰 SISTEMA DE CONTROLE FINANCEIRO

Esse projeto foi desenvolvido com o objetivo de praticar Java e Spring Boot na criação de uma aplicação web completa para controle financeiro pessoal.

A ideia foi sair um pouco da teoria e construir algo útil de verdade, simulando um sistema que poderia ser usado no dia a dia para acompanhar gastos e entradas.

🚀 TECNOLOGIAS UTILIZADAS:
 -Java
 -Spring Boot
 -Thymeleaf
 -Chart.js
 -HTML / CSS
 -Maven

📌 FUNCIONALIDADES:
 -Cadastro de gastos e entradas
 -Edição e remoção de registros
 -Visualização de um resumo financeiro geral
 -Gráficos de gastos por categoria
 -Gráfico de evolução financeira por mês
 -Filtro de gastos por período

🎯 OBJETIVO DO PROJETO:
O principal objetivo foi aprender na prática como funciona o desenvolvimento de uma aplicação web com Spring Boot, passando por:

 -Estruturação de backend
 -Integração com banco de dados
 -Renderização de páginas com Thymeleaf
 -Exibição de dados em gráficos
